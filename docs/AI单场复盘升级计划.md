# AI单场复盘升级计划

日期：2026-03-07

## 目标

围绕“单场直播”先做一个视觉冲击强、老板一眼能看懂、且能够自然收费的复盘模块。

本期明确不做：

- 多场次对比
- 自动日报/周报
- 批量复盘
- 复杂告警
- 重型 AI 数据管线

核心原则：

1. 第一版先把界面价值和感知价值做出来。
2. 先做“看起来很值钱”的单场复盘，而不是先做最重的算法。
3. 先用现有数据拼出老板视角，再逐步增加 AI 深度。

## 推荐定位

推荐产品名：`AI单场复盘`

副标题可用：

- 运营总监视角的直播结论卡
- 给老板看的直播复盘摘要
- 一场直播的结果、问题与下一步

## 逐步升级路线

### V1：唬人版单场复盘（优先上线）

目标：

- 强视觉
- 强结论
- 强老板感知
- 实现轻

建议页面结构：

1. 顶部总览区
   - `本场复盘评分`（大号分数卡，例如 78/100）
   - `一句话总结`
   - `本场状态标签`（稳、强、波动、大哥驱动、互动偏弱等）

2. 中部图表区
   - `礼物流速图`（按时间段）
   - `互动热度图`（弹幕/点赞/进房）
   - `在线波动趋势图`（如数据不足可先用已有时段指标替代）
   - `关键时刻时间轴`（高峰、掉人、冷场、爆点）

3. 下部结果卡片区
   - `本场亮点`
   - `主要问题`
   - `下一场建议`
   - `本场价值客户分析`

其中“本场价值客户分析”建议拆成三类：

- `核心价值客户`：本场送礼高，且历史总价值高
- `潜力转化客户`：本场互动高，但送礼低或首次送礼
- `流失风险客户`：历史高价值，但本场明显变弱

V1 的实现建议：

- 图表和标签优先用规则引擎生成
- AI 只负责把结构化结果润色成“老板能看懂的话”
- 页面风格做得像高端数据智能产品，而不是普通后台列表

### V1.5：增强版单场复盘

在 V1 基础上补 3 个更“唬人”的功能点：

1. `价值客户九宫格`
   - 按“本场贡献 × 历史价值”分布
   - 一眼看出谁是本场大哥、谁是潜力客户、谁在掉线

2. `直播节奏诊断`
   - 输出“起场 / 过渡 / 爆发 / 回落”四阶段判断
   - 让复盘更像专业运营视角，而不是静态报表

3. `建议动作卡`
   - 对不同客户层给动作建议
   - 例如：值得私聊、值得维持、值得拉群、值得二次触达

### V2：真正的 AI 单场复盘

在前两版验证点击率和付费意愿后，再增加更重的 AI：

- 更完整的摘要生成
- 关键时刻原因解释
- 话术/节奏建议
- 主播行为建议
- 互动问题原因拆解

到这个阶段，再考虑把“单场复盘”从数据增强页，升级成独立的高价值收费能力。

## 第一版最值得做的“唬人功能点”

建议从下面 6 个中选 4~5 个即可：

1. `本场复盘评分`
2. `一句话老板摘要`
3. `礼物高峰时间轴`
4. `互动热度图`
5. `关键时刻卡片`
6. `本场价值客户分析`

这 6 个里，最推荐必做的是：

- 本场复盘评分
- 一句话老板摘要
- 关键时刻卡片
- 本场价值客户分析

因为它们最容易被截图、汇报、发老板、发客户。

## 本场价值客户分析建议

这是单场复盘里最容易产生真实业务价值的一块。

建议第一版只输出 3 类客户，不要太复杂：

### 1. 核心价值客户

定义建议：

- 本场送礼金额进入 Top N
- 且历史总价值达到一定门槛

输出字段建议：

- 昵称
- 本场贡献
- 历史总价值
- 常来房间
- 建议动作

### 2. 潜力转化客户

定义建议：

- 本场发言活跃 / 互动活跃
- 本场贡献不高，但历史价值不高或首次出手

输出重点：

- 值不值得重点引导成交
- 适合哪种跟进方式

### 3. 流失风险客户

定义建议：

- 历史总价值高
- 本场明显变弱，或只来不出手

输出重点：

- 提醒主播/运营重点留意
- 给出维护动作建议

## 定价与扣点建议

## 当前点数价格

当前 AI 点数包价格约为：

- `10 点 / ¥10` → `¥1.00/点`
- `100 点 / ¥50` → `¥0.50/点`
- `500 点 / ¥200` → `¥0.40/点`

所以真实成交视角下，主流点价大致可按 `¥0.4 ~ ¥0.5/点` 估算。

## 为什么单场复盘不能太便宜

如果一个直播间每天 `1~2` 场，一个月大概 `30~60` 场。

若单场只扣：

- `3 点` → 月成本约 `90~180 点` → 折合约 `¥36~¥90`
- `5 点` → 月成本约 `150~300 点` → 折合约 `¥60~¥150`

这相对于 `¥599~¥1699/月` 的订阅价格，确实偏轻，容易让“单场复盘”显得不够高级，也不利于拉高 AI 收入。

## 推荐扣点梯度

### 方案A：轻量老板摘要

- 内容：一句话总结 + 关键时刻 + 简单客户卡
- 建议价格：`6~8 点 / 场`

适合：

- 当前阶段的第一版
- 用较轻实现做出高感知价值

### 方案B：标准单场复盘

- 内容：评分 + 关键图表 + 亮点问题 + 价值客户分析 + 建议动作
- 建议价格：`10~12 点 / 场`

适合：

- 作为主推收费版本
- 与订阅价形成更自然的价格带

### 方案C：高阶单场复盘

- 内容：更完整 AI 解释、更强建议、更细客户分层
- 建议价格：`15~20 点 / 场`

适合：

- 后续高级版或企业版能力
- 不建议作为第一版默认扣点

## 当前推荐

基于当前点数价格与产品阶段，建议：

- V1 唬人版单场复盘：`8 点 / 场`
- 如果带“价值客户分析 + 建议动作卡”：`10 点 / 场`
- 已生成内容再次查看：`0 点`
- 强制重新生成：按完整扣点再次收费

这样换算后：

- `30 场 / 月` → `240~300 点` → 约 `¥96~¥150`
- `60 场 / 月` → `480~600 点` → 约 `¥192~¥300`

这个区间比“几十点一个月”更健康，也更符合“老板复盘工具”的定位。

## 最终建议

如果只做一个最合适的第一版，建议就是：

- 名称：`AI单场复盘`
- 核心卖点：`老板摘要 + 关键时刻 + 本场价值客户分析`
- 界面重点：分数卡、趋势图、时间轴、客户卡片
- 扣点建议：`8~10 点 / 场`

这会比“只做一个轻量文本摘要”更能唬人，也比“上来做完整多场智能分析”更现实。
